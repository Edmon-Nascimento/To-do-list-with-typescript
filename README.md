# Task List (React + TypeScript)

A simple task list application built with React and TypeScript. The app allows users to add, edit, complete, and delete tasks, with data persistence using LocalStorage.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed or uncompleted
- Delete tasks
- Persist tasks using LocalStorage
- Automatically focus the input when editing a task

## Technologies Used

- React
- TypeScript
- React Hooks
  - useState
  - useEffect
  - useRef
  - useCallback
- LocalStorage
- CSS
